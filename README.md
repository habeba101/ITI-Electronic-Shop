# ITI Electronic Shop

A front-end e-commerce web application developed as part of the ITI training program.
This project simulates an online electronics store where users can browse products, add items to their cart, and manage their favorite products using client-side JavaScript.

# Overview

ITI Electronic Shop is a static e-commerce prototype built using HTML, CSS, and JavaScript.
The goal of this project is to demonstrate:
DOM manipulation
Dynamic rendering using JavaScript
Multi-page navigation
Shopping cart logic handling
Favorites management system
The application focuses entirely on front-end functionality without a backend or database.

# Features

Browse electronic products
Add products to cart
Add products to favorites
Remove items from cart or favorites
Display total price in cart
Dynamic UI updates without page reload
